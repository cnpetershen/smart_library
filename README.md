# smart_library
The Smart Library is a library background management system built on Spring Boot 3.1, Vue 3 and Element Plus. The backend adopts Spring Security with JWT authentication, MyBatis-Plus, MySQL and Redis. The frontend is developed with Vue 3, TypeScript, Pinia and ECharts. Its core modules cover book management, borrowing and returning management, reader management, user authentication, as well as AI intelligent analysis powered by the DeepSeek API, which supports functions including borrowing trend analysis, popularity ranking, category statistics and personalized recommendation.
